Pokemon
By Zachary Francis

To run the the game, follow these instructions:
1. Install PIP with help from the following link.
    https://pip.pypa.io/en/stable/installing/
2. After PIP is installed, use the console to install "Pillow" using sudopip
3. Be sure that the "get-pip.py" is in the folder labeled "Pokemon"
4. Now, in the Pokemon folder, simply run the file called "PokemonFinal.py"
    using Python 3
5. Sit back and enjoy the game
